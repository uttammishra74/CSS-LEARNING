# 🎨 3D Neumorphism UI Clone

A modern **3D Neumorphism UI Clone** built using **HTML5** and **CSS3**. This project recreates popular neumorphic UI components with realistic shadows, smooth hover animations, and clean minimal styling.


---

## 🚀 Features

* ✨ Modern Neumorphism (Soft UI) Design
* 🎯 3D Buttons with Hover & Click Effects
* ❤️ Interactive Icon Buttons
* 🔍 Neumorphic Search Bar
* 🔒 Password Field Design
* 📌 Follow Button
* 🎚️ Custom Toggle Switches
* 📱 Clean Responsive Layout
* ⚡ Pure HTML & CSS (No JavaScript)

---

## 🛠️ Built With

* HTML5
* CSS3
* SVG Icons
* Font Awesome
