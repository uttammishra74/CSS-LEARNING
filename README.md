# 🛍️ Shopsy Clone

A frontend clone of the **Shopsy** homepage built using **HTML5** and **CSS3**. This project was created to practice modern web development concepts such as Flexbox, layout design, responsive sections, navigation bars, and UI recreation.

## ✨ Features

- 🏠 Modern navigation bar
- 🔍 Search bar with search icon
- 👤 Login button with Font Awesome icon
- 🛒 Cart section
- 🏪 Become a Seller option
- 📦 Category cards (Children, Men, Women, Shoes, Beauty)
- 🎯 Promotional banner section
- 🖼️ Product image sections
- 🎨 Clean and simple UI inspired by Shopsy

## 🚀 How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/Shopsy-Clone.git
```

2. Open the project folder.

```bash
cd Shopsy-Clone
```

3. Open `CLone.html` in your browser.

Or simply use **Live Server** in Visual Studio Code.
